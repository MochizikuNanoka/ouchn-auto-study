# 国开学习平台自动刷课助手

> 面向国家开放大学（电大中专）学习平台的自动学习脚本：自动播放视频、配合爱问答助手自动交卷，并支持断点续传、异常恢复和课程目录重新扫描。

## 适用平台

| 平台 | 网址 | 状态 |
|------|------|------|
| 电大中专 | `zydz-menhu.ouchn.edu.cn` | 已验证 |

## 功能

| 功能 | 说明 |
|------|------|
| 自动播放视频 | 解析课程总览页进度，跳过已完成内容；收到视频结束事件后完整等待 10 秒再进入下一步，99% 兜底同样会倒计时确认。 |
| 自动交卷 | 检测爱问答助手完成答题后，依次交卷、确认、查看试卷并返回课程页。 |
| 异常恢复 | 不按 500、服务器错误或 AxiosError 等页面文本刷新；仅在课程目录、播放器或题目状态等待超时未就绪时 F5。刷新前保存断点，刷新后重新扫描课程目录，不复用旧目录。 |
| 断点续传 | 使用浏览器本地存储（`ouchn_autoplay_v2`）按课程、章节、节次和任务类型恢复。 |
| 清缓存重置 | 只清除助手自己的 `ouchn_autoplay_*` 缓存，停止任务并重新加载课程目录；不会退出平台登录。 |
| 目录诊断 | 等待章节与课程项稳定渲染后才解析，并输出中文诊断日志。 |
| 顺序纠错 | 平台提示“请先完成 X.X”时，自动匹配并导航到对应节次。 |

## 安装

1. 在浏览器中安装 [Tampermonkey](https://www.tampermonkey.net/) 扩展。
2. 打开 Tampermonkey 管理面板，选择“添加新脚本”。
3. 将 [国开学习平台-自动刷课助手.user.js](./国开学习平台-自动刷课助手.user.js) 的全部内容粘贴进去。
4. 按 `Ctrl+S` 保存。

## 使用

1. 登录学习平台并进入课程总览页。
2. 在右上角控制面板点击“开始”。
3. 脚本自动展开章节、扫描未完成节次并依次处理。

遇到目录异常、旧断点异常或需要从头扫描时，点击“清缓存重置”。它不会清除登录状态；页面重新加载后回到课程总览页，再点击“开始”即可。

> 考试答题依赖爱问答助手插件，完成判定基于答题卡 `.everyAnswer.AnswerEnd` 类名。
>
> 从 v2.0.1 或更早版本升级时，旧断点没有课程 ID。为避免跨课程误续跑，请回到对应课程总览页点击一次“开始”，脚本会建立新断点；之后 F5 可自动恢复。

## 控制面板

| 按钮 | 功能 |
|------|------|
| 开始 | 启动自动学习。 |
| 暂停 / 继续 | 暂停或恢复当前循环。 |
| 停止 | 停止任务并清除当前断点。 |
| 清缓存重置 | 清除助手缓存并重新加载当前课程目录，不退出登录。 |
| 调试 | 展开更新检查区域。 |

## 技术原理

### DOM 索引稳定定位

早期版本通过标题文本定位节次，但课程页面可能有多个同名视频或考试。当前版本在解析时记录 DOM 索引，并在回到课程页后按索引定位，减少同名节次漂移。

```text
CourseModel.buildModel()
  -> expandAllChapters()       展开章节级面板
  -> 遍历 .el-collapse-item    按页面内容区分视频与考试
  -> 构建层级模型              章节 -> 节次对 -> 视频/考试
  -> getPendingTasks()         展平为待处理任务，每项保留 DOM 索引
  -> navigateToDomIndex()      按全局索引定位，不依赖标题文本
```

### 刷新后的全新目录扫描

自动 F5 前只保存当前任务指针、统计数据和重试信息，不保存完整课程目录。刷新时脚本会跳转到带唯一扫描标识的课程总览路由；恢复后先等待新 DOM 稳定、重新建立任务列表，再用保存的任务指针匹配继续位置。

```json
{
  "version": 4,
  "courseId": "3016",
  "chapterIdx": 0,
  "pairIdx": 2,
  "itemType": "exam",
  "title": "1.3 文件权限管理",
  "totalTasks": 49,
  "stats": { "videos": 15, "exams": 10, "errors": 0, "skipped": 0 },
  "retryCount": 0,
  "autoResume": true,
  "timestamp": 1751689200000
}
```
