# 国开学习平台 自动刷课助手

> 国家开放大学（电大中专）学习平台自动刷课脚本 — 自动看视频、配合爱问答助手自动交卷

## 适用平台

目前已测试通过：

| 平台 | 网址 | 状态 |
|------|------|------|
| 电大中专（中央广播电视中等专业学校） | `zydz-menhu.ouchn.edu.cn` | ✅ 可用 |

其他国开体系平台理论上兼容，欢迎反馈。

## 功能

- **自动看视频** — 解析课程总览页进度，跳过已完成节次，自动播放未完成视频
- **自动交卷** — 检测爱问答助手答题完成后，自动点击交卷 → 确认 → 查看试卷 → 返回
- **500 重试** — 页面出错自动 F5 刷新，指数退避最多 5 次
- **断点续传** — 进度存 localStorage，页面刷新 / 崩溃后自动恢复
- **99% 保底** — 视频进度卡 99% 时等待 10 秒后强制完成
- **调试模式** — 可单独测试视频或考试节次

## 安装

1. 浏览器安装 [Tampermonkey](https://www.tampermonkey.net/) 扩展
2. 打开 Tampermonkey 管理面板 → **添加新脚本**
3. 将 `国开学习平台-自动刷课助手.user.js` 全部内容粘贴进去
4. `Ctrl+S` 保存

## 使用

1. 登录学习平台，进入**课程总览页**：
   ```
   https://zydz-menhu.ouchn.edu.cn/learningPlatform/#/myCourse/study?id=3098
   ```
2. 右上角出现控制面板，点击 **开始**
3. 脚本自动：
   - 展开所有章节 → 扫描全部节次
   - 跳过已完成的（进度 100%）
   - 逐个处理未完成节次
   - 每节完成后 F5 刷新防止页面超时

> ⚠️ 需要配合**爱问答助手**插件（考试答题用）。考试检测逻辑基于答题卡 `.everyAnswer.AnswerEnd`。

## 控制面板

| 按钮 | 功能 |
|------|------|
| 开始 | 启动自动刷课 |
| 暂停 / 继续 | 暂停或恢复 |
| 跳过 | 跳过当前节次 |
| 停止 | 停止并清除进度 |
| D | 展开调试面板（可单独测试视频/考试） |

## 流程

```
课程总览页
 ├── 展开所有章节 + 扫描节次
 ├── 过滤已完成的（进度 = 100%）
 └── 对每个未完成节次：
      ├── 视频 → 点击 → F5(如需) → 播放 → 进度 ≥99% + 等10s → 完成
      ├── 考试 → 点击 → 等爱问答助手完成 → 自动交卷 → 返回
      └── 完成后 F5 刷新页面 → 继续下一节
```
